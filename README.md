# Cricket Scorecard Management System

## Overview

A console-based Real-Time Cricket Scorecard Management System developed in C++ using Object-Oriented Programming principles.

The project simulates a cricket match with ball-by-ball scoring, batting and bowling statistics, fantasy points, player injuries, catch ratings, and detailed match summaries.

## Features

* Toss simulation
* Pitch selection
* Ball-by-ball scoring
* Batting scorecards
* Bowling scorecards
* Extras tracking (Wides, No Balls, Byes, Leg Byes)
* Catch rating system
* Injury handling
* Fantasy points calculation
* Maiden over detection
* Hat-trick detection
* Man of the Match selection
* Best Catch of the Match
* Top performers summary
